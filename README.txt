BEATZ BRAWL V8

Open index.html in Chrome or Edge.

New in V8:
- Replaced the generated title screen with title_screen.png
- Added menu_theme.mp3 for the title screen
- Added character_select_theme.mp3 for fighter select
- Fight screen still uses fight_theme.mp3
- M mutes/unmutes all music and sound effects

Controls:
A / D = move
W = jump
S = crouch
B = block
F = punch
G = kick
H = special
M = mute
P = pause
R = restart round

Browser note: music can only start after a key press or tap because of browser autoplay rules.


V9: Title screen now uses two-step audio unlock. First tap/key starts menu music and stays on title; second Enter/tap goes to character select.


V13: Added Gary as selectable fighter, character select cards with highlight/confirm, Gary fire special effect, and fixed round/match end so only Enter or the on-screen continue button advances.

V14: adjusted Gary sprite scaling so punch/special/block no longer shrink compared with standing pose.
